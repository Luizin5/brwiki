# Brwiki

A library to scrap articles from wikipedia

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'brwiki'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install brwiki

## Usage

```ruby

include Brwiki
puts Brwiki::search("en","juice")  
```
